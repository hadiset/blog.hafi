# Task: Improvisasi Landing Page Baskara Wedding Organizer

Project ini menggunakan **Astro + Tailwind CSS**. Tidak ada framework component (React/Vue/Svelte).
Sebelum mengerjakan, baca seluruh file yang relevan terlebih dahulu agar perubahan konsisten dengan kode yang sudah ada.
Jangan ubah struktur atau styling yang tidak disebutkan dalam task ini.

---

## Task 1 — Perbaiki Mobile Menu

Mobile menu saat ini sudah menggunakan JS untuk toggle class. Pastikan behavior berikut berjalan dengan benar:

- Tombol hamburger hanya muncul di mobile (`md:hidden`)
- Saat tombol diklik, mobile menu muncul dengan transisi smooth (opacity + translate)
- Saat diklik lagi atau klik di luar area menu, menu kembali tersembunyi
- Tidak boleh menggunakan library JS eksternal — cukup vanilla JS atau Astro script tag
- Pastikan tidak ada konflik class antara state mobile dan desktop

| Field | Tipe | Keterangan |

| Email | `input type="email"` | Opsional |
